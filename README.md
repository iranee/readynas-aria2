# Readynas Aria2 Applications

Aria2是一个命令行下运行、多协议、多来源下载工具（HTTP/HTTPS、FTP、BitTorrent、Metalink），内建 XML-RPC 用户界面

### 安装程序
* 1， 下载以 **.deb** 结尾的安装程序
* 2， 进入ReadyNAS管理界面，点进 **应用程序**
* 3， 选择上传并安装应用程序
![安装图解](https://github.com/iranee/readynas-aria2/blob/master/install.png)
